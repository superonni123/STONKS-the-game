## Info
Made with Python 3.8.5 
## Setup
Clone or download this repo to desktop for example
```
$ cd ../STONKS-the-game
```
if you have python installed
```
$ python main.py
```
else to launch the .exe file
```
$ cd ../STONKS-the-game/dist/main/main.exe
```
